# unhexo

`unhexo` is a free and open-source hex-edit software.

Primary development occurs on [Codeberg.org](https://codeberg.org/igorzizinio/unhexo), but there's also a [Github](https://github.com/igorzizinio/unhexo) repository

## Features

- [ ] hex file reading
- [ ] tabs
- [ ] compare files
- [ ] workspaces
- [ ] theming
