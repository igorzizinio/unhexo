# unhexo

`unhexo` is a free and open-source hex-edit software.

## Features

- [ ] hex file reading
- [ ] tabs
- [ ] compare files
- [ ] workspaces
- [ ] theming

## Design inspirations

- Visual Studio Code
- Obsdian
- Notion

## Thinking

Deixar em memória os arquivos abertos (tabs), porém armazenar no Rust, o web só deve receber o que realmente vai ser exibido.
